# dsa_utilities

<h3> In this repository, you will find implementations of various data structures and algorithms in golang programming languages. The code is organized into folders based on the type of data structure or algorithm. </h3>

## How to run the codebase

1. Clone the repository to your local machine using the command: `git clone`
2. Navigate to the project directory: `cd dsa_utilities`
3. Create a `main.go` file using the command: `touch main.go` and import the required package from the folder you want to run
4. Run the code using the command: `go run <main.go>`
